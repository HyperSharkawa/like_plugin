# 麦麦赞我插件（like_plugin）

为麦麦提供点赞功能的插件。

## 安装

需要在 NapCat 中的网络配置添加一个 HTTP 服务器。如果不会填，建议Host填`0.0.0.0`，Port填`9999`，其余保持默认。

插件安装完成后，重启1次麦麦以生成默认配置文件，然后根据下方的配置项说明进行配置。

## 配置项说明

### `like_plugin` 配置段

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `enable_rewrite_reply` | `bool` | `true` | 是否启用使用人设重写回复。启用后，点赞结果会通过生成器重写成更自然的回复 |
| `enable_splitter` | `bool` | `false` | 是否启用长回复分段发送。启用后，回复会被分段发送 |

### `napcat` 配置段

| 配置项 | 类型 | 默认值 | 说明                                                    |
|--------|------|------|-------------------------------------------------------|
| `host` | `str` | `127.0.0.1` | NapCat 中 Http 服务的地址                                   |
| `port` | `int` | `9999` | NapCat 中 Http 服务的端口号                                  |
| `token` | `str` |  | NapCat 中 Http 服务的 Token（可选）。如果 NapCat 中填写了此项，则此处也必须填写 |

## 使用

在聊天中发送以下任意命令即可触发点赞功能：

```
/like
#like
/赞我
#赞我
/麦麦赞我
#麦麦赞我
赞我
麦麦赞我
```
